hcSsn
directive in module hcUiApp

Description

Overview

'hcSsn' directive provides three different input boxes to fill the user's three parts of the Social Security Number respectively in these input boxes.

Dependencies

  • templateURL
  • ngModel
  • hcSsn

Usage

as element:
<hc-ssn>
</hc-ssn>

Directive info

  • This directive creates new scope.
