hcDate
directive in module hcUiApp

Description

Overview

'hc-date' directive provides three selection inputs for months, days, and years values. User will get a drop-down list and can select one option for each of the values to get a required date.

Dependencies

  • templateURL
  • ngModel

Usage

as attribute
<ANY hc-date>
   ...
</ANY>

Directive info

  • This directive creates new scope.
