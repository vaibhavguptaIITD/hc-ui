hcGreaterThanEqual
directive in module hcUiApp

Description

Overview

'hcGreaterThanEqual' directive compares two input elements. If first input is greater than or equal to the second input, then the validity will be true otherwise validity will be false.

Usage

as attribute
<input type="text" hc-greater-than-equal>
   ...
</input type="text">
