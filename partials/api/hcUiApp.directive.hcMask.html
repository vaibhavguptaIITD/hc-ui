hcMask
directive in module hcUiApp

Description

Overview

'hcMask' directive is used for masking. Here we are using this directive for 4 types of data inputs i.e. date of birth, zip, SSN and phone no.

Dependencies

  • ngModel

Usage

as attribute
<input type="text" hc-mask>
   ...
</input type="text">

Directive info

  • This directive creates new scope.
